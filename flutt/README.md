# Lava Jump

Простая офлайн-игра на Flutter. Стикмен поднимается прыжками между стенами,
обходит шипы и убегает от лавы.

## Управление

- Коснитесь экрана у стены, чтобы прыгнуть к другой стене.
- Один раз в воздухе можно коснуться экрана повторно и изменить направление.
- Каждый прыжок или воздушный разворот даёт одно очко.
- Касание шипа или лавы завершает игру.

## Запуск

```bash
cd flutt
flutter pub get
flutter run
```

Если платформенные папки отсутствуют, выполните `flutter create .` один раз.

## Проверка

```bash
dart format . --set-exit-if-changed
flutter analyze
flutter test
```
